# 模拟考试小程序 - 实施计划

## 项目概述
开发一款网页版模拟考试小程序，支持上传表格题库、随机抽题练习、错题集功能。纯前端应用，数据存储在浏览器本地（localStorage），无需后端服务。

## 技术选型
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **状态管理**: Zustand
- **路由**: React Router DOM
- **表格解析**: SheetJS (xlsx) — 用于解析上传的 Excel/CSV 题库文件
- **图标**: lucide-react
- **数据持久化**: localStorage（错题集、题库缓存）

## 题库文件格式约定
上传的表格文件（.xlsx / .xls / .csv）需包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 题目 | 题干内容 | "下列哪项是..." |
| 选项A | A选项内容 | "选项A文本" |
| 选项B | B选项内容 | "选项B文本" |
| 选项C | C选项内容 | "选项C文本" |
| 选项D | D选项内容 | "选项D文本" |
| 答案 | 正确答案 | "A" / "B" / "C" / "D" |

> 支持判断题（仅选项A="对"、选项B="错"），也支持单选题。

## 页面与功能设计

### 页面1：首页（题库管理）
- **上传题库区域**: 拖拽或点击上传 Excel/CSV 文件
- **题库信息展示**: 显示当前已加载的题库名称、题目总数
- **开始练习按钮**: 随机抽取100道题进入练习模式
- **错题集入口**: 跳转至错题练习页面
- **清空题库按钮**: 移除已加载的题库数据

### 页面2：练习页面
- **题目展示区**: 显示当前题号/总题数、题干、四个选项
- **选项交互**: 点击选项即选中，高亮显示
- **提交判断**: 选中后点击"确认"按钮判断对错
  - 正确：选项变绿，显示"回答正确"
  - 错误：选中项变红，正确答案变绿，显示"回答错误，正确答案是 X"
- **导航功能**: 上一题/下一题按钮，题号快速跳转
- **答题卡**: 侧边或底部答题卡，显示每题状态（未答/正确/错误）
- **练习结果**: 做完所有题后显示成绩统计（总题数、正确数、错误数、正确率）
- **错题自动记录**: 答错的题自动加入错题集（localStorage持久化）

### 页面3：错题集页面
- **错题列表**: 显示所有错题，含题干、你的答案、正确答案
- **错题练习**: 可从错题集中随机抽题练习（流程同练习页面）
- **错题统计**: 显示错题总数
- **清除错题**: 可单题移除或全部清空
- **答对自动移除**: 错题练习中答对的题目自动从错题集中移除

## 核心流程

```
用户上传题库 → 解析Excel → 存入状态 → 随机抽100题 → 逐题练习 → 判断对错 → 错题入集 → 查看结果
                                                                    ↓
                                                              错题集练习 → 答对移除 → 更新错题集
```

## 数据模型

### Question（题目）
```typescript
interface Question {
  id: string;
  content: string;
  options: { A: string; B: string; C: string; D: string };
  answer: 'A' | 'B' | 'C' | 'D';
}
```

### WrongQuestion（错题记录）
```typescript
interface WrongQuestion extends Question {
  wrongAnswer: 'A' | 'B' | 'C' | 'D';
  addedAt: number;
}
```

### PracticeState（练习状态）
```typescript
interface PracticeState {
  questions: Question[];
  currentIndex: number;
  answers: Record<string, 'A' | 'B' | 'C' | 'D'>;
  results: Record<string, 'correct' | 'wrong'>;
}
```

## Zustand Store 设计

### useQuestionStore（题库状态）
- `questions: Question[]` — 全部题目
- `loadFromFile(file: File): Promise<void>` — 解析上传文件
- `clearQuestions(): void` — 清空题库
- `getRandomQuestions(count: number): Question[]` — 随机抽题

### useWrongQuestionStore（错题集状态）
- `wrongQuestions: WrongQuestion[]` — 错题列表
- `addWrongQuestion(q: WrongQuestion): void` — 添加错题
- `removeWrongQuestion(id: string): void` — 移除错题（答对时）
- `clearWrongQuestions(): void` — 清空错题集
- 持久化到 localStorage

### usePracticeStore（练习状态）
- `questions: Question[]` — 当前练习的题目
- `currentIndex: number` — 当前题号
- `answers: Record<string, string>` — 用户作答
- `results: Record<string, 'correct' | 'wrong'>` — 判断结果
- `selectAnswer(id: string, answer: string): void` — 选择答案
- `submitAnswer(id: string): void` — 提交判断
- `nextQuestion(): void` / `prevQuestion(): void` — 导航
- `resetPractice(): void` — 重置练习

## 路由定义

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 题库上传与管理 |
| `/practice` | 练习页 | 随机抽题练习 |
| `/practice/wrong` | 错题练习 | 从错题集练习 |
| `/wrong` | 错题集 | 查看错题列表 |

## UI 设计风格
- **整体风格**: 清新简约，以学习/教育为主题
- **主色调**: 深蓝 (#1e40af) + 翠绿 (#059669) 作为正确色 + 红色 (#dc2626) 作为错误色
- **背景**: 浅灰渐变，带微妙纹理
- **卡片式布局**: 题目区域使用圆角卡片，带柔和阴影
- **字体**: 使用 Noto Sans SC 作为中文字体
- **按钮**: 圆角按钮，带 hover 过渡动画
- **答题卡**: 网格布局，不同状态用不同颜色标识
- **响应式**: 桌面优先，适配移动端
